**"Описание работы программы 'Скатерть самобранка':

Главное меню :
Программа начинает работу с главной страницы, где пользователь видит анимированный заголовок "Скатерть самобранка" (цвет: синий, движение: из стороны в сторону через CSS-анимацию).
Под заголовком находится подсказка: "Проси что хочешь - материализую что хочешь."
Анимация на заднем фоне :
На заднем фоне программы есть 50 маленьких круглых точек (диаметр 10px, цвет оранжевый, анимация: увеличение/уменьшение размера через CSS), они рассредоточены по всей странице и их как будто ветром перемещает по странице.
Точки зациклены и постоянно меняют свои размеры, создавая эффект "сбора и разбора".
Форма для ввода запроса :
Пользователь видит форму для ввода запроса на материализацию товара:
Текстовая подсказка: "Что материализовать:".
Поле для ввода текста (placeholder: "Введите запрос...").
Кнопка "Сгенерировать" (зелёного цвета).
Имитация сложной работы :
При нажатии на кнопку "Сгенерировать" или нажатии клавиши Enter:
Появляется анимация работы системы в виде последовательных статусов в одной строке, текст заменяет друг друга:
"Материализация начата..." (цвет: оранжевый, появляется через 2-4 секунды).
"Готовим материал..." (цвет: оранжевый, заменяет предыдущий статус через 2-4 секунды).
"Очень сложная работа..." (цвет: оранжевый, заменяет предыдущий статус через 2-4 секунды).
"Почти всё готово..." (цвет: оранжевый, заменяет предыдущий статус через 2-4 секунды).
"Материализация завершена!" (цвет: зелёный, остаётся на экране).
Форма для доставки :
После завершения материализации появляется форма для указания адреса доставки:
Текстовая подсказка: "Адрес:".
Поле для ввода адреса (placeholder: "Введите ваш адрес куда доставить заказ...").
Текстовая подсказка: "Выберите удобное для Вас время доставки заказа курьером (это абсолютно бесплатно!):".
Выпадающий список для выбора времени доставки (временные интервалы: "10:00", "12:00", "14:00", "16:00", "18:00", "20:00", "22:00", "23:00").
Кнопка "Отправить курьера" (зелёного цвета).
Анимация процесса доставки :
После выбора адреса и времени доставки:
Появляется анимация процесса доставки в виде последовательных статусов:
"Ищем свободного курьера..." (цвет: оранжевый, появляется через 2-4 секунды).
"Отправляем заказ..." (цвет: оранжевый, заменяет предыдущий статус через 2-4 секунды).
"Готово!" (цвет: оранжевый, заменяет предыдущий статус через 2-4 секунды).
После показа всех статусов доставки:
Показывается финальное сообщение:
"Ожидайте курьера в назначеное время..." (цвет: зелёный).
"Время доставки: [выбранное время]" (цвет: зелёный).
"Спасибо, что воспользовались нашим сервисом!" (цвет: зелёный).
Анимация замены статусов :
Все статусы заменяют друг друга в одной строке через CSS-анимацию (появление нового текста).
Цвет статусов меняется в зависимости от их содержимого:
Для статусов "Материализация завершена!" и "Готово!" — зелёный (#28a745).
Для остальных статусов — оранжевый (#ff6f00).
Полная логика работы :
Если пользователь вводит пустой запрос, показывается предупреждение: "Введите запрос!".
Если пользователь вводит пустой адрес, показывается предупреждение: "Введите адрес доставки заказа!".
Все статусы работы системы и процесса доставки заменяют друг друга последовательно, создавая эффект "динамической работы".
Интерфейс программы наглядно показывает каждый этап работы системы и процесса доставки.
Финальное сообщение :
После успешной доставки, пользователь видит финальное сообщение:
"Ожидайте курьера в назначеное время...".
"Время доставки: [выбранное время]".
"Спасибо, что воспользовались нашим сервисом!".
Интерфейс :
Все этапы работы системы и процесса доставки отображаются через текстовые сообщения и анимации.
Все статусы работы системы заменяют друг друга через CSS-анимацию.
Цветовая смена статусов и финального сообщения происходит через JavaScript.
Задний фон :
Задний фон содержит анимацию точек, которые зацикленно собираются и разбираются, создавая эффект "оранжевой сети"." и перемещаются по странице.
